Rolling Rhino Remix - Rhino Documentation

Rhino is a command-line utility specifically created for Rolling Rhino Remix. There are currently two functions that Rhino can perform:

  • rhino-init
  • rhino-update

The first command, rhino-init, will initialise Rolling Rhino Remix. It will replace /etc/apt/sources.list with the Ubuntu devel repositories (as opposed to the next Ubuntu release). Once it has completed this step it will automatically update all other packages via APT. After rhino-init has been completed you will not be required to utilise it again.

The second command, rhino-update, will be the command that you will use to update your system. It will update your system via apt as well as provide the latest tools that we develop and the latest Linux kernel direct from the Ubuntu mainline repository. This tool was developed so that we are able to easily provide updates to the Rolling Rhino Remix desktop, as well as provide the latest Linux kernel binaries. It is recommended that you use this command as opposed to apt update && apt upgrade that is more common on standard Ubuntu desktops.
