Ubuntu Rolling Remix - Bug Tracker

  • Please be aware that Ubuntu Rolling Remix is a rolling release distribution. Bugs are to be expected and it will not be perfectly stable. This is also amplified by the fact that Ubuntu is not a rolling release distribution by nature. While we will attempt to prevent bugs as much as possible there is no guarentee.

List of known bugs:

  • Issue performing rhino-upgrade with Nvidia hardware
    • There is currently a known issue where if you run rhino-upgrade on hardware with Nvidia your system will become unrecoverable. We are aware of this issue and are looking into ways to fix it. We currently believe this issue is due to an issue where the Nvidia driver does not work due to the fact that the Kernel has been upgraded to an unsupported version. We are looking into ways for fixing this before the next release. For now it is recommended to run sudo apt update && sudo apt upgrade as opposed to rhino-upgrade

File a bug report:

If you have found a bug in Ubuntu Rolling Remix please feel free to report it to us.

To report a bug please send us an email

  • Email address: ubunturollingremix@gmail.com
  • Subject Line: Bug Report
  • Body: Please describe your issue in detail and the steps to reproduce it in the body.
